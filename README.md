# Game of Thrones Character Survival Prediction

Проект посвящен анализу данных и прогнозированию выживаемости персонажей из вселенной Game of Thrones. На основе признаков о персонажах, их семье, происхождении и социальных характеристиках строится модель, которая предсказывает, жив ли персонаж в книге.

В ходе проекта были выполнены следующие этапы:

- анализ структуры данных и пропущенных значений;
- исследование распределений признаков;
- создание новых признаков на основе исходных данных;
- кодирование категориальных переменных;
- подготовка данных для обучения модели;
- обучение и оценка модели бинарной классификации;
- формирование предсказаний для тестовой выборки.

## Цель

Предсказать, выживет ли персонаж книги Game of Thrones.

Целевая переменная:
- isAlive — признак того, жив ли персонаж

## Данные

В работе используются два набора данных:
- train — обучающая выборка;
- test — тестовая выборка.

В данных содержатся признаки о персонаже, например:
- пол;
- дата рождения;
- семья;
- супруг;
- отец;
- мать;
- наследник;
- статус родственников;
- популярность персонажа;
- другие характеристики.

## Что сделано

### 1. Предобработка данных
- проанализированы пропуски;
- удалены или преобразованы неинформативные признаки;
- созданы дополнительные бинарные и категориальные признаки;
- выполнено кодирование категориальных переменных.

### 2. Исследовательский анализ данных
- изучено распределение целевой переменной;
- проанализированы числовые и категориальные признаки;
- рассмотрено влияние признаков на выживаемость персонажа.

### 3. Моделирование
- данные подготовлены для обучения;
- обучена модель классификации;
- выполнена оценка качества;
- сформированы предсказания для тестовой выборки.
